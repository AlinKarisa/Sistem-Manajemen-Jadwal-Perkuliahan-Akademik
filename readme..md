# Deskripsi Program

JADWALKU merupakan aplikasi berbasis bahasa Go (Golang) yang digunakan untuk membantu pengelolaan jadwal perkuliahan akademik. Program ini dirancang untuk memudahkan pengguna dalam melakukan manajemen data jadwal kuliah seperti menambahkan, mencari, mengubah, menghapus, dan menampilkan data jadwal perkuliahan.

Program dibuat sebagai implementasi materi Algoritma dan Pemrograman 2 dengan memanfaatkan:
- Array
- Struct
- Procedure
- Function
- Searching
- Sorting
- Sequential Search
- Binary Search
- Selection Sort / Insertion Sort

---

# Fitur Program

- Menambahkan data jadwal kuliah
- Menampilkan seluruh data jadwal
- Mengubah data jadwal
- Menghapus data jadwal
- Mencari data jadwal
- Mengurutkan data jadwal
- Menu interaktif berbasis terminal

---

# Struktur Data

Data jadwal perkuliahan terdiri dari:
- Nama Mata Kuliah
- Kode Mata Kuliah
- Hari
- Jam
- Ruangan
- Dosen Pengampu
